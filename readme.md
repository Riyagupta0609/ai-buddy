# 🎓 AI Learning Buddy Riya

An AI-powered learning assistant built using **Streamlit** and **Google Gemini API**. It helps students understand concepts through simple explanations, real-life examples, quizzes, and interactive question answering.
## ✨ Features

* 📖 Explain concepts in simple language.
* 🌍 Provide real-life examples for better understanding.
* 📝 Generate multiple-choice quizzes with answers.
* 💬 Ask anything and get AI-powered responses.
* 🎓 Beginner-friendly explanations.
* ⚡ Fast and interactive user interface built with Streamlit.

---

## 🛠️ Technologies Used

* Python
* Streamlit
* Google Gemini API
* Google Generative AI SDK
* Pyngrok (for running the app from Google Colab)

## 🚀 Installation

1. Clone the repository.

```bash
git clone https://github.com/your-username/AI-Learning-Buddy-Riya.git
```

2. Move into the project folder.

```bash
cd AI-Learning-Buddy-Riya
```

3. Install the required packages.

```bash
pip install -r requirements.txt
```

4. Add your Google Gemini API key in the application (or configure it securely using environment variables or Streamlit Secrets).

5. Run the application.

```bash
streamlit run app.py
```
